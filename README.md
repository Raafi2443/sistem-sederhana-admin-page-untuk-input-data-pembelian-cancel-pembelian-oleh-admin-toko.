# sistem-sederhana-admin-page-untuk-input-data-pembelian-cancel-pembelian-oleh-admin-toko.
Ini adalah aplikasi admin toko sederhana untuk mengelola produk, stock, pembelian, serta melihat dan mengedit data secara langsung.

Fitur Utama

Input pembelian barang
Cancel pembelian
Tambah produk baru
Edit harga dan stock produk
Hapus produk
Lihat riwayat pembelian
Lihat riwayat perubahan harga & stock (termasuk tambah/hapus produk)
Hapus semua riwayat pembelian atau perubahan
Tampilan stock real-time di halaman input pembelian

Persyaratan Sistem

Node.js (versi 14 atau lebih baru)
MySQL (bisa menggunakan XAMPP, Laravel Valet, atau server MySQL lainnya)
Browser modern (Chrome, Firefox, dll)
